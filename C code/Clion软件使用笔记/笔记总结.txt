setbuf函数用于打开和关闭缓冲机制，关闭缓冲区的功能。
void setbuf ( FILE * stream, char * buffer );
设置流缓冲区
指定流用于 I/O 操作的缓冲区，该缓冲区将成为完全缓冲的流。或者，如果 buffer 为 null 指针，则对流禁用缓冲，该流将成为无缓冲流。

一旦流与打开的文件关联，但在对它执行任何输入或输出操作之前，应调用此函数。
